# ChiVO Web Page

## Requeriments
 * Ruby >= 2.0.0
 * Rails >= 4

## Installation
    git clone git@github.com:ChileanVirtualObservatory/web.git
    bundle install
    rails server

then visit `http://localhost:3000`

## APP

to see the branch practice, take a look `http://chivo-practica.herokuapp.com/`
